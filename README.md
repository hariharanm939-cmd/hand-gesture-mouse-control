# Hand Gesture Controlled Mouse System

> Control your computer using hand gestures — no physical mouse required.

## 🚀 Features
- Real-time hand tracking  
- Cursor movement using gestures  
- Gesture-based click actions  
- Contactless interaction  

## 👨‍💻 My Contributions
- Implemented hand tracking using OpenCV and MediaPipe  
- Mapped gestures to mouse controls  
- Optimized for real-time performance  

## 🛠 Tech Stack
Python, OpenCV, MediaPipe
